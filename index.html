Sobre o Projeto

Este site foi desenvolvido com foco na acessibilidade para pessoas com deficiências motoras e visuais.

Recursos de Acessibilidade

  • Teclas de atalho para navegação.
  • Texto com contraste adequado para pessoas com baixa visão.
  • Compatibilidade com leitores de tela.
